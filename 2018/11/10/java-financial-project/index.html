4.统一错误处理

https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#boot-features-error-handling
spring boot提供了默认的/error全局错误处理
对于machine clients(程序，jar包，http client ,ajax发起的), it produces a JSON response 包括HTTP status, the exception message.
对于浏览器（browser clients会自动添加accept: text/html到报文头）发起的请求renders the same data in HTML format 如果用postman加上Accept:text/html就返回html
通过实现ErrorController接口注册到容器中或者添加一个ErrorAttributesbean
还可以用@ControllerAdvice拦截controller的错误
