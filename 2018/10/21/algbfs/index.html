倒水问题 BFS ax + by = m 最大公约数

365 容量x,y的两个杯子能否量出z

Input: x = 3, y = 5, z = 4
Output: True
Input: x = 2, y = 6, z = 5
Output: False

bfs 4种情况

-2*3 + 2*5 = 4
倒掉3的容器2次倒满5的容器2次
1.装满5，倒给3 -> (0,2)
2.装满5，倒给3 -> (4,3)

只要z是x,y的最大公约数的倍数就True

  • 裴蜀定理:ax + by = m
    有整数解，当且仅当 m是d = gcd(a,b)的倍数

d 是最小的 可以写成ax+by形式的正整数。
ax+by = 1有整数解 当且仅当 a,b互质

ax+by = (a+b)/2
第一个瓶子倒入x次第二个倒出y次
倒进倒出都是大瓶子，倒进1次之后继续用还要把小瓶子倒回大瓶子，倒出同理。
但是最后 一定是放在两个小瓶子里不用倒回大瓶子 所以操作数=(a+b)/gcd(a,b)-1
